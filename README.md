# Software CPP-ROC-SORT

Aplicação em Python + Streamlit para executar o método CPP-ROC-SORT.

## Como instalar

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Como rodar

```bash
streamlit run app.py
```

## Arquivos obrigatórios

- dados/avaliacoes_exemplo.csv
- dados/rankings_exemplo.csv
- dados/perfis_exemplo.csv
- dados/variancias_exemplo.csv

Arquivo opcional:

- dados/pesos_decisores_exemplo.csv
